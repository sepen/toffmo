# toffmo

The OFFice MOtivator

## Requirements

PyGTK (http://www.pygtk.org)

## Installation

Build and install
```
$ make
$ sudo make install
```

## Usage
```
  toffmo <options>

  Where options are:
  -h, --help           Show this help information
  -V, --version        Show version information
  --conf=CONFIG        Use alternate config file
```
